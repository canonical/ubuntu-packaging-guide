Tutorial
********

This section contains step-by-step tutorials to help you get started with
Ubuntu packaging and development. We hope the tutorials make as few assumptions
as possible and are accessible to anyone with an interest in Ubuntu packaging.

This should be a great place to start learning about packaging and development.

-----

Core tutorial
=============

This tutorial will introduce you to the basics of Ubuntu packaging, while
helping to set up your computer so that you can start working with packages.
